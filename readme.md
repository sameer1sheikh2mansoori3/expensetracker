## Run Locally

Clone the project

```bash
  git clone https://github.com/sameer1sheikh2mansoori3/expensetracker.git
```

Go to the project directory

```bash
  cd backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
